## 主張

クランク=ニコルソン法の離散化式に含まれるn+1時刻の隣接点値をガウス=ザイデル反復（逐次代入）で更新し続けることで、行列を陽に組むことなく収束解を得られる。

## 展開

クランク=ニコルソン法の更新式の右辺には未知量である `temp[i+1]`, `temp[i-1]`（n+1時刻）が含まれるため、本来は連立1次方程式を行列形式で解く必要がある。しかしガウス=ザイデル法では、格子点を順に走査しながら直前に更新した値をその場で利用し、残差 `resd` が閾値（例: 1×10⁻⁶）以下になるまで反復することで収束させる。コードでは内側ループが最大 `itermax=1000` 回反復し、早期収束すれば `break` で抜ける。この実装方針はコードの単純さを保ちつつ、三重対角行列ソルバー（Thomas法）より実装コストが低い。

## 具体例

```python
for iter in range(itermax):
    resd = 0.0
    for i in range(1, nx-1):
        tp = temp[i]
        temp[i] = 1/(1+ck) * ((1-ck)*temp_old[i]
            + 0.5*ck*(temp_old[i+1]+temp_old[i-1])
            + 0.5*ck*(temp[i+1]+temp[i-1]))
        resd += abs(temp[i] - tp)
    if resd <= eps:
        break
```
