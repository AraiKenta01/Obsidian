## 主張
Gitには`.git`フォルダの実体をリポジトリ外に分離する機能がある。
## 展開
Obsidianのvaultをicloudとgitの共同で管理する際に、icloudが`.git`フォルダまで同期対象にしてしまい不具合が頻発した。[[複数の同期システムを同時使用してはいけない]]に記載の通り、2システムによる共同管理はトラブルの元である。
そこで解決手段を調べた結果、どうやら`.git`フォルダを別の場所に配置することができることを知った。
正直使いどころは限定的だと思うが、知らなかったし面白い機能だと思う。
## 具体例
`.git`フォルダはどこにおいても問題ない。
管理対象リポジトリのルートに`.git`という名前のテキストファイルを作成し、`.git`フォルダの絶対パスを記述する。
```
gitdir: /Users/hoge/Documents/fuga/.git
```
